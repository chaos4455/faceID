# Face ID


## Requirements
- Ubuntu 16.04
- Python 3.5
- OpenCV
- Keras
- TensorFlow(backend)


## Getting Started
Creating virtualenv
```bash
$ cd faceID
$ virtualenv env --python=python3.5
$ source env/bin/activate
```

Install Dependencies
```bash
$ pip install -r requirements.txt
```

Run
```bash
$ python main.py
```
